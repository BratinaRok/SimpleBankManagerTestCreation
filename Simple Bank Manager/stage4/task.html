Description

At this stage we will implement a feature to calculate the exchange from one currency to another. When user successfully login, user should be able to go to Calculate Exchange screen, user can select the currencies to convert from and to, enter an amount to convert and trigger the conversion by clicking a button. The converted amount will be displayed in a text view.


Objectives

  1. Add a Fragment named CalculateExchangeFragment to the navigation graph;
    • Add an TextView to CalculateExchangeFragment with id calculateExchangeLabelFromTextView, the text should be convert from;
    • Add an TextView to CalculateExchangeFragment with id calculateExchangeLabelToTextView, the text should be convert to;
    • Add an Spinner to CalculateExchangeFragment with id calculateExchangeFromSpinner, the spinner should contain this text in dropdown EUR, GBP, USD;
    • Add an Spinner to CalculateExchangeFragment with id calculateExchangeToSpinner, the spinner should contain this text in dropdown EUR, GBP, USD;
    • Add an TextView to CalculateExchangeFragment with id calculateExchangeDisplayTextView, this textView will be used to show calculated amount to 2 decimal points;
    • Add an EditText to CalculateExchangeFragment with id calculateExchangeAmountEditText. It should accept input type numberDecimal and have a hint Enter amount;
    • Add an Button to CalculateExchangeFragment with id calculateExchangeButton and with a text calculate;
  2. On the design tab of nav_graph make a connection from UserMenuFragment to CalculateExchangeFragment. The code generated after making the connection should contain an action with id action_userMenuFragment_to_calculateExchangeFragment and destination calculateExchangeFragment;
  3. After click on userMenuExchangeCalculatorButton display CalculateExchangeFragment. Users should be able to choose from which currency and to which currency to convert to using calculateExchangeFromSpinner and calculateExchangeToSpinner. Input amount to convert at calculateExchangeAmountEditText, and confirm conversion with button click calculateExchangeButton;
  4. Test may pass the exchangeMap as a Serializable to MainActivity through intent.extras. Retrieve that balance, using the key "exchangeMap". You will want to cast it back to a map to be able to use it as a map. The type signature of exchangeMap is Map<String, Map<String, Double>>. In case no exchangeMap is passed use 

    
val defaultMap = mapOf(
    "EUR" to mapOf(
        "GBP" to 0.5,
        "USD" to 2.0
    ),
    "GBP" to mapOf(
        "EUR" to 2.0,
        "USD" to 4.0
    ),
    "USD" to mapOf(
        "EUR" to 0.5,
        "GBP" to 0.25
    )
)
    as default value for exchangeMap. The outer key of exchangeMap is the source currency from which we are converting and the inner key is the target currency to convert to;
  5. If user selects same currency on both calculateExchangeFromSpinner and calculateExchangeToSpinner, then show a Toast message Cannot convert to same currency and replace currency at calculateExchangeToSpinner with the next available currency;
  6. If no value is passed to calculateExchangeAmountEditText and button calculateExchangeButton is clicked it should show a Toast message Enter amount;
  7. While at CalculateExchangeFragment clicking on back button should redirect users back to UserMenuFragment;

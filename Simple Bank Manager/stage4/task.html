Description

At this stage we will implement a feature to calculate one currency to another. When user successfully login, user should be able to go to Calculate exchange screen, user can select the currencies to convert from and to, enter an amount to convert and trigger the conversion by clicking a button. The converted amount will be displayed in a text view.


Objectives

  1. Add a Fragment named CalculateExchangeFragment to the navigation graph;
    • Add an TextView to CalculateExchangeFragment with id calculateExchangeLabelFromTextView, the text should be convert from;
    • Add an TextView to CalculateExchangeFragment with id calculateExchangeLabelToTextView, the text should be convert to
    • Add an Spinner to CalculateExchangeFragment with id calculateExchangeFromSpinner, the spinner should contain this text in dropdown eur, gbp, usd.
    • Add an Spinner to CalculateExchangeFragment with id calculateExchangeToSpinner, the spinner should contain this text in dropdown eur, gbp, usd.
    • Add an TextView to CalculateExchangeFragment with id calculateExchangeDisplayTextView, this textView will be used to show calculated amount to 2 decimal points.
    • Add an EditText to CalculateExchangeFragment with id calculateExchangeAmountEditText. It should accept input type numberDecimal and have a hint Enter amount.
    • Add an Button to CalculateExchangeFragment with id calculateExchangeButton and with a text calculate.
    • On the design tab of nav_graph make a connection from UserMenuFragment to CalculateExchangeFragment. The code generated after making the connection should contain an action with id action_userMenuFragment_to_calculateExchangeFragment and destination calculateExchangeFragment;
    • After click on userMenuExchangeCalculatorButton display CalculateExchangeFragment. Display on calculateExchangeFromSpinner dropdown menu currencies eur, gbp, usd. Display on calculateExchangeToSpinner dropdown menu currencies eur, gbp, usd. At the calculateExchangeFromSpinner and calculateExchangeToSpinner , user should be able to choose from which currency and to which currency to convert to. Input amount to convert at the calculateExchangeAmountEditText, and confirm conversion with button click calculateExchangeButton.
      • We will use fixed exchange rates that was actual at that time. Conversion rates are :
    • 1 EUR = 0.87 GBP
    • 1 EUR = 1.075 USD
    • 1 GBP = 1.14 EUR
    • 1 GBP = 1.14 USD
    • 1 USD = 1.00 EUR
    • 1 USD = 0.877 GBP
    • If user selects at calculateExchangeFromSpinner same currency that is already selected at calculateExchangeToSpinner, it should show a Toast message Cannot convert to same currency and replace currency at calculateExchangeToSpinner with the next available currency.
    • If no value is passed to calculateExchangeAmountEditText and button calculateExchangeButton is clicked it should show a Toast message Enter amount.
    • While at CalculateExchangeFragment clicking on back button should redirect users back to UserMenuFragment

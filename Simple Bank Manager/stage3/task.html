Description

After successful login user should be able to transfer money to or from account and view current balance at the balance view.

The navigation graph will help us to implement the navigation between the screen of our app.

Objectives

  1. Add a Fragment named ViewBalanceFragment to the navigation graph
  2. Add an TextView to ViewBalanceFragment with id viewBalanceTextBalanceTextView, the text should be "Balance:"
  3. Add an TextView to ViewBalanceFragment with id viewBalanceShowBalanceTextView, with hint "0.0$". This TextView will be used to show account balance.
  4. On the design tab of nav_graph make a connection from UserMenuFragment to ViewBalanceFragment. The code generated after making the connection should contain an action with id action_userMenuFragment_to_viewBalanceFragment and destination userMenuFragment
  5. After a successful login users should be redirected to UserLoginFragment. To make the navigation you can use findNavController().navigate(R.id.action_userMenuFragment_to_viewBalanceFragment). It is also possible to pass a Bundle as second argument of navigate method to pass arguments to userMenuFragment, which you can later retrieve with arguments?.getString(...)
  6. All requirements from stage1 and stage2 are still valid, including toast messages
